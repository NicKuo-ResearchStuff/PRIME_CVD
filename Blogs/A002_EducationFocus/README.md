## Series 02 — Education Focus

<img src="https://github.com/NicKuo-ResearchStuff/PRIME_CVD/blob/main/Supporting_Images/ZFig_PRIME_CVD_Series02_Logo.png" width="600"/>

* **Data Asset 1: Designing Socioeconomic Stratification Assignments with a Clean CVD Cohort** (see [here](https://github.com/NicKuo-ResearchStuff/PRIME_CVD/tree/main/Blogs/A002_EducationFocus/SubBlog01))</br>
  → A guide for educators on using the clean synthetic cohort to design fair, reproducible assessment tasks that test students’ ability to reason about socioeconomic gradients and modelling readiness.

* **Data Asset 1: Understanding Baseline Patients and Cox Models with a Clean CVD Cohort** (see [here](https://github.com/NicKuo-ResearchStuff/PRIME_CVD/tree/main/Blogs/A002_EducationFocus/SubBlog02))</br>
  → A teaching guide that uses the clean cohort to explain how Cox models define and interpret a baseline patient, and why coding decisions shape real-world risk estimation.

* **Data Asset 2: Designing EMR Reconstruction Assignments with Presence-Only Diagnosis Data** (see [here](https://github.com/NicKuo-ResearchStuff/PRIME_CVD/tree/main/Blogs/A002_EducationFocus/SubBlog03))</br>
  → A guide for educators on using the synthetic EMR tables to design assessment tasks that require students to harmonise diagnoses and construct defensible prevalence estimates from presence-only data.
